Git est un système de contrôle de version distribué (VCS, Version Control System) largement utilisé dans le développement de logiciels. Il a été créé par Linus Torvalds en 2005. Git permet de gérer et de suivre les modifications apportées à un ensemble de fichiers au fil du temps

GitHub est une plateforme web d'hébergement de code source et de collaboration pour les développeurs de logiciels. Elle offre un espace où les développeurs peuvent stocker, gérer et collaborer sur des projets de développement logiciel à l'aide du système de contrôle de version Git. GitHub est devenu l'une des plateformes les plus populaires pour le développement collaboratif de logiciels open source et propriétaires.
